#Description:

Real-time to do app.
The app allows the users to create and manage thier tasks, add subtasks and add users to collaborate together on a task. The app is updated in real time.



##Technologies: 

* MEAN stack (Node, Express, Angular, MogoDb) 
* mogoose 
* socket.io
* bootstrap
* toastr
* html
* css
* body-parser
* google fonts
* font-awesome.
